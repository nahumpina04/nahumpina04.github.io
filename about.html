Projects

I've been involved in several projects. At the College of Engineering, I've been able to create robots, programming them using block code and C++. A notable project is the Biofeedback Glove. In this project, I was given a robotic arm and tasked with creating a glove, circuit, and program to interconnect all of the tasks and make the given arm move, repsoding to the glove. In my group, I was tasked to complete the programming. I created an simple program that would begin with a calibration phase to adapt the movements as close to the user's hand as possible. The program would then continuously read changes in the glove and output them to the arm. Due to malfuntioning sensors, the arm didn't work adequately. However, when testing with fully functioning sensors, the arm was able to move accordingly.
